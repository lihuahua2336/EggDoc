## 安装

选择你的安装方式和操作系统，复制命令后粘贴到终端运行。

<EggAiClaudeCodeConfig client:load />

## 验证安装

安装结束后运行：

```sh
claude --version
```

看到版本号即表示安装完成。

默认模式调用 Anthropic 推荐的原生安装器，不修改 Claude Code 配置。选择 EggAi 配置后，脚本会保留原有设置并写入 EggAi 网关配置。

## 中国大陆网络说明

EggAi 配置会先验证 EggAi 的 Anthropic Messages 网关，再写入 `settings.json`；Claude Code 使用 `ANTHROPIC_AUTH_TOKEN` 访问 EggAi，不要求 Anthropic 账户登录可用。官方安装器下载仍是独立链路，无法访问 `claude.ai` 时请设置 `CLAUDE_CODE_INSTALLER_URL` 到经过审核的可达镜像，或通过系统 PowerShell 代理运行。页面部署可以设置 `PUBLIC_CLAUDE_CODE_INSTALLER_ORIGIN`，让生成命令按平台自动带上正确的镜像脚本。

页面生成命令来自 `PUBLIC_INSTALLER_ORIGIN`。面向中国用户部署时，应把它配置为可访问的自定义域名，并在该域名上提供 EggDoc 安装脚本。安装成功只代表 Claude Code 二进制和 EggAi 网关配置可用，不代表官方账户登录可用。
