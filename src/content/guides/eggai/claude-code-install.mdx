## 安装

选择你的安装方式和操作系统，复制命令后粘贴到终端运行。

<EggAiClaudeCodeConfig client:load />

## 验证安装

安装结束后运行：

```sh
claude --version
```

看到版本号即表示安装完成。

默认模式调用 Anthropic 推荐的原生安装器，不修改 Claude Code 配置。选择 EggAi 配置后，脚本会保留原有设置并写入 EggAi 网关配置。

[查看 Anthropic 官方安装文档](https://code.claude.com/docs/en/setup#install-claude-code)
