## macOS、Linux 或 WSL

在终端运行：

```sh
curl -fsSL 'https://eggdoc.pages.dev/install/claude-code.sh' | sh
```

WSL 用户需要在 WSL 终端中运行这条命令。

## Windows

在 PowerShell 运行：

```powershell
irm 'https://eggdoc.pages.dev/install/claude-code.ps1' | iex
```

无需以管理员身份运行 PowerShell。

## 验证安装

安装结束后运行：

```sh
claude --version
```

看到版本号即表示安装完成。

脚本调用 Anthropic 推荐的原生安装器，不修改 Claude Code 配置，也不会自动完成账号登录。原生安装会由 Claude Code 在后台检查更新。

[查看 Anthropic 官方安装文档](https://code.claude.com/docs/en/setup#install-claude-code)
