## macOS、Linux 或 WSL

在终端运行：

```sh
curl -fsSL https://claude.ai/install.sh | bash
```

## Windows

在 PowerShell 运行：

```powershell
irm https://claude.ai/install.ps1 | iex
```

## 验证安装

安装结束后运行：

```sh
claude --version
```

看到版本号即表示安装完成。
