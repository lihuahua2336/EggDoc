## 安装

选择你的安装方式和操作系统，复制命令后粘贴到终端运行。

## 验证安装

安装结束后运行：

```sh
claude --version
```

看到版本号即表示安装完成。

脚本会先检测 Node.js 22 和 npm。缺失或版本过低时，macOS、Linux 和 WSL 从 `nodejs.org` 下载并校验 Node 22，Windows 通过精确 winget 包 `OpenJS.NodeJS.LTS` 自动安装。随后脚本通过 `https://registry.npmmirror.com` 安装 Anthropic 官方 npm 包 `@anthropic-ai/claude-code`，并把 npm 全局命令目录加入 PATH。npm 镜像只用于本次安装命令，不会永久修改用户 npm 配置。默认模式不修改 Claude Code 配置；选择 EggAi 配置后，脚本会保留原有设置并写入 EggAi 网关配置。

## 中国大陆网络说明

EggAi 配置会先验证 EggAi 的 Anthropic Messages 网关，再写入 `settings.json`；Claude Code 使用 `ANTHROPIC_AUTH_TOKEN` 访问 EggAi，不要求 Anthropic 账户登录可用。Node.js 官方渠道或 npm 镜像不可访问时，脚本会明确报错并停止，不会自动切换来源；PowerShell 使用系统代理配置，Shell 透传标准代理环境变量。

页面生成命令来自 `PUBLIC_INSTALLER_ORIGIN`，默认值为 `https://doc.eggai.icu`。安装成功只代表 Claude Code 二进制和 EggAi 网关配置可用，不代表官方账户登录可用。
