Cherry Studio 支持通过 OpenAI 兼容接口连接第三方模型服务。EggAi 可以作为其中一个 API 提供方接入。

## 配置入口

在 Cherry Studio 中添加新的模型服务提供方，并选择 OpenAI 兼容类型。

## 需要填写的信息

- API Key
- Base URL
- 模型名称

保存后，可以先用一个简短问题测试连接是否正常。
