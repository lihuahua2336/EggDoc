这篇指南用于记录在 Cursor 中配置 EggAi 的完整步骤。

## 准备工作

开始之前，你需要准备：

- EggAi API Key
- EggAi Base URL
- 已安装的 Cursor

## 配置步骤

1. 打开 Cursor 设置。
2. 找到模型或 API 配置区域。
3. 填入 EggAi 提供的 Base URL。
4. 填入 API Key。
5. 保存配置并发送一次测试请求。

## 常见问题

如果请求失败，优先检查 Base URL 是否多写了路径、API Key 是否复制完整、当前模型名称是否可用。
