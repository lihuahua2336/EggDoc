这一节会把 EggDoc 自身作为项目实战，从内容模型、页面骨架到 Cloudflare Pages 部署逐步完成。

## 项目目标

- 使用 Astro 管理静态站点
- 使用 Markdown 和 MDX 管理内容
- 使用 shadcn/ui 建立克制、清晰的界面
- 使用 Pagefind 添加静态搜索

> 这篇文章使用 MDX，是为了后续演示可复用的交互组件。
