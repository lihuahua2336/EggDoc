# Adapt the EggAi ecosystem contract

EggDoc integrates with the same Logto resource audience, ecosystem scopes, and `/api/ecosystem/me`, `/api/ecosystem/models`, and `/api/ecosystem/tokens` contract used by Infinite Canvas. An EggDoc-owned adapter translates that external contract into EggAi API Account and EggAi API Credential concepts so tutorial UI and configuration templates do not depend directly on New API response field names.
