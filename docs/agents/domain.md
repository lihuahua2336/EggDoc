# Domain Docs

EggDoc is a single-context repository.

## Before exploring

- Read the root `CONTEXT.md` for the project's canonical domain language.
- Read relevant decisions under `docs/adr/` before changing the affected area.
- Proceed without warning if a referenced domain document does not exist.

## Vocabulary

Use terms as defined in `CONTEXT.md` in PRDs, issues, tests, code, and reviews. Do not substitute words listed under `_Avoid_`.

If a required domain term is missing, use the domain-modeling flow to resolve it rather than silently inventing competing language.

## Decisions

Surface conflicts with existing ADRs explicitly. Do not silently override an architectural decision.
